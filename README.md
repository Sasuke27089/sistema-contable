# Sistema Contable (inicio)

Pequeño sistema contable con autenticación y módulo de usuarios.

Requisitos
- Node.js 18+ / npm

Instalación
```bash
cd /workspaces/sistema-contable
npm install
```

Iniciar servidor
```bash
npm start
# luego abrir http://localhost:3000
```

Notas
- El proyecto usa SQLite en `data/database.sqlite3`.
- Las vistas están en `views` (EJS) y Tailwind se incluye por CDN para estilo rápido.
